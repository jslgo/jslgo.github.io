this is it in JavaScript

发表于 2018-07-01 | 分类于 JavaScript

¶this是什么

       函数是语句的容器, 而this代表函数调用invocation时的上下文环境context

¶this从哪里来

       每个函数默认会接收两个附加的参数, this和arguments

¶this的5种应用场景

  1. 函数调用
           当函数不作为宿主全局Global对象(浏览器里为window, node.js里为global)以外的对象的方法来调用时, this指向全局对象

  2. 方法调用
           当函数作为宿主全局Global对象(浏览器里为window, node.js里为global)以外的对象的方法来调用时, this指向该方法的拥有者对象

  3. 构造器调用
           当函数被new调用时, this指向连接到该函数原型对象的新对象

  4. 转移上下文调用
           apply, call, bind是函数的方法, 会将函数执行时的this指向作为参数传入的对象

  5. 箭头函数调用
           箭头函数中的this指向定义时所在的对象, 而不是使用时所在的对象
